# GitHub SAML Auto-Continue (Safari)

A userscript that automatically clicks the "Continue" button on GitHub's SAML SSO page for the **oslokommune** organization. Same functionality as the [Chrome extension](https://github.com/ivaruf/auto-saml-clicker), but for Safari.

## Prerequisites

Install a userscript manager for Safari. Either of these free options work:

- [**Userscripts**](https://apps.apple.com/app/userscripts/id1463298887) (free, open source, recommended)
- [**Tampermonkey**](https://apps.apple.com/app/tampermonkey/id1482490089) (free)

## Installation

1. Install one of the userscript managers above from the App Store
2. Enable it in **Safari > Settings > Extensions**
3. Open the file `github-saml-auto-continue.user.js` in Safari — the userscript manager will prompt you to install it
4. Confirm the installation

That's it. The SAML "Continue" screen will now be skipped automatically.
